### <b>Run tests ğŸ§ª</b><a name="run-tests"></a>

How should it be tested? 
To run tests, run the following command:

- HTML linter errors run:

```
npx hint . --fix
```

- CSS linter errors run:

```

- JavaScript linter errors run:

```
npx eslint . --fix
```

- For unit testing, run:

```
